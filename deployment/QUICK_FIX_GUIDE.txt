OPTION C: Fix Dashboard Code (Advanced)

If you need to support multiple column names:

1. SSH to server:
   ssh racing-server

2. Edit the dashboard code:
   cd /home/tactical/racing-dashboard
   nano src/dashboard/app.py

3. Find line ~626 (search for "vehicle_number"):
   vehicles = sorted(df['vehicle_number'].unique()) if 'vehicle_number' in df.columns else []

4. Replace with this flexible version:

   # Find vehicle column with multiple possible names
   vehicle_col = None
   for col in ['vehicle_number', 'Vehicle', 'VehicleNumber', 'car', 'Car', 'CarNumber']:
       if col in df.columns:
           vehicle_col = col
           break

   vehicles = sorted(df[vehicle_col].unique()) if vehicle_col else []

5. Save (Ctrl+X, Y, Enter)

6. Restart dashboard:
   sudo systemctl restart racing-dashboard

7. Test upload again
