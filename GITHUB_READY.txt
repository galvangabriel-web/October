TROUBLESHOOTING
---------------

Problem: Push rejected (non-fast-forward)
Solution:
- Review changes: git log --oneline -10
- Force push: git push -u origin main --force
- Or pull first: git pull origin main --rebase

Problem: File too large
Solution:
- Check: git ls-files -s | sort -k4 -n -r | head -10
- Use Git LFS if models > 100MB
- Verify .gitignore is excluding data files

Problem: Too many files
Solution:
- Check what's staged: git status
- Verify .gitignore is working
- Remove unwanted files: git rm --cached <file>
